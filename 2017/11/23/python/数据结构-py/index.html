sort(_*_,key=None,reverse=False)

This method modifies the sequence in place for economy of space when sorting a large sequence. To remind users that it operates by side effect, it does not return the sorted sequence (usesorted()to explicitly request a new sorted list instance). # 对list本身产生影响，

Thesort()method is guaranteed to be stable. A sort is stable if it guarantees not to change the relative order of elements that compare equal — this is helpful for sorting in multiple passes (for example, sort by department, then by salary grade). 稳定排序

字典排序

字典排序主要还是使用sorted函数，关键是key参数的配置配合keys,values(),items()

也可以使用OrderDict类

sorted(dict.keys(), key=abs)

sorted(dict.items(), key=lambda r:r[0])

key参数可以考虑使用operator.itemgetter函数

如果你传入多个索引参数给itemgetter()，它生成的callable对象会返回一个包含所有元素值的元组， 并且sorted()函数会根据这个元组中元素顺序去排序。 但你想要同时在几个字段上面进行排序（比如通过姓和名来排序，也就是例子中的那样）的时候这种方法是很有用的。

collections.ChainMap

chainmap对多干个映射对象进行link

假设有多个dict对象，需要判断某key是否存在（可能存在其中任何一个dict）,而多个dict是实时更新的

这时使用ChainMap比对多个dict进行update操作更加方便，chainmap对象实时映射原dict

具体使用方法和dict一样，注意的是对chainmap进行字典增删只会影响chainmap的第一个dict

试图删除第一个dict中不存在的key会raise KeyError

ChainMap.maps 以list对象返回所有的dict(这是属性，不是函数）
