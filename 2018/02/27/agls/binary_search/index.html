35. Search Insert Position

Given a sorted array and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

You may assume no duplicates in the array.

Example 1:
Input: [1,3,5,6], 5
Output: 2
Example 2:
Input: [1,3,5,6], 2
Output: 1
Example 3:
Input: [1,3,5,6], 7
Output: 4
Example 4:
Input: [1,3,5,6], 0
Output: 0

low bound 注意循环终止条件(剩下最后两个item时)和异常处理（数组为空）

34. Search for a Range

Given an array of integers sorted in ascending order, find the starting and ending position of a given target value.

Your algorithm’s runtime complexity must be in the order of O(log n).

If the target is not found in the array, return [-1, -1].

For example,
Given [5, 7, 7, 8, 8, 10] and target value 8,
return [3, 4].

配合low bound 和 high bound（找到第一个等于target和最后一个等于target的下标）
注意：当low bound未找到target，即返回-1时，high bound也必然找不到，可直接返回

bisetc

bisetc是一个用使用二分搜索的模块（built-in），维护一个有序列表
提供两种操作：
参数a为已经拍好序的list对象

  • 查找有序列表中插入某值时的index
    bisect.bisect_left(a, x, lo=0, hi=len(a)) # 前置查询
    bisect.bisect_right(a, x, lo=0, hi=len(a)) # 后置查询
    bisect.bisect(a, x, lo=0, hi=len(a)) # 后置查询

  • 在有序列表中插入某值
    bisect.insort_left(a, x, lo=0, hi=len(a)) # 前置插入
    bisect.insort_right(a, x, lo=0, hi=len(a)) # 后置插入
    bisect.insort(a, x, lo=0, hi=len(a)) # 后置插入

虽然查询是O(logn)复杂度，但是插入运算依然是O(n)
